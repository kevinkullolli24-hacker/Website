Çfarë është një website?

Një website është një koleksion faqesh interneti të lidhura me njëra-tjetrën, të cilat mund të hapen përmes një shfletuesi (si Chrome, Safari, ose Firefox). Websitet përdoren për të shpërndarë informacion, për të shitur produkte, për të komunikuar, ose thjesht për t'u argëtuar.

Për të ndërtuar një website nga e para, ju duhen 3 teknologji bazë të cilat punojnë gjithmonë së bashku:

HTML

Struktura (Skeleti)

HTML përdoret për të krijuar "skeletin" e një website-i. Ajo përcakton ku do të vendosen tekstet, imazhet dhe lidhjet (links) në faqe.

Mëso HTML

CSS

Dizajni (Pamja)

CSS përdoret për t'i dhënë jetë dhe ngjyra website-it tuaj. Ajo kontrollon ngjyrat, fontet, dhe pozicionimin e saktë të elementeve.

Mëso CSS

JavaScript

Interaktiviteti (Truri)

JavaScript i jep jetë website-it tuaj, duke mundësuar animacione, butona funksionalë, dhe ndryshime të të dhënave pa rifreskuar faqen.
