Çfarë është JavaScript?

Truri dhe Interaktiviteti i Web-it

JavaScript është gjuha e programimit e cila i jep jetë faqeve në web. Nëse HTML është skeleti dhe CSS është pamja e jashtme, JavaScript është sistemi nervor, energjia elektrike dhe uji – ai e bën ndërtesën të funksionojë vërtet!

Në thelb, JavaScript u lejon website-ve të reagojnë ndaj veprimeve të përdoruesit, të dërgojnë e të marrin të dhëna në prapaskenë, dhe të krijojnë ndërfaqe komplekse e ndërvepruese pa pasur nevojë të ngarkoni përsëri faqen (refresh).

Pse na duhet JavaScript?

Këto janë disa nga funksionet kryesore që bëjnë JavaScript një nga gjuhët më të rëndësishme sot:

Interaktivitet

Mundëson dritaret e vogla ("pop-up"), klikimin e butonave (si për shembull butoni jonë për ndryshimin e modalitetit Dritë/Errësirë) dhe menutë lëvizëse.

Animacione Komplekse

Ndërsa CSS mund të krijojë animacione bazë, JavaScript përdoret për ndryshime dinamike me llogaritje apo krijimin e lojërave vizuale direkt në browser.

Përditësimi i të Dhënave

Ju lejon të komunikoni me databaza dhe të merrni të dhëna nga jashtë (si p.sh. parashikimi i motit) duke i shfaqur ato menjëherë te përdoruesi.

Një Shembull i Thjeshtë në JavaScript

// Ky kod gjen një buton në faqe dhe kur klikohet, tregon një mesazh. let butoni = document.getElementById("butoni-im"); butoni.addEventListener("click", function() { alert("Ju sapo klikuat butonin me sukses!"); });
