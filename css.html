Çfarë është CSS?

Gjuha që i jep stil dhe bukuri Web-it

CSS (Cascading Style Sheets) është gjuha që përdoret për t'i dhënë dizajn, ngjyra dhe strukturë vizuale një dokumenti HTML. Nëse HTML është "skeleti" dhe muret e një ndërtese, CSS është bojat, dekorimet e brendshme, dritat dhe arredimi.

Me CSS, ju keni mundësinë të kontrolloni plotësisht ngjyrat, madhësinë e shkrimit, pozicionin e elementeve në ekran, hijet (shadows), dhe madje edhe animacionet komplekse.

Karakteristikat Bazë të CSS

Këto janë tre konceptet kryesore që formojnë bazën e çdo stilimi në CSS:

Sfondi dhe Ngjyrat

Mund të ndryshoni ngjyrën e tekstit ose të shtoni ngjyra e imazhe në sfondin (background) e elementeve tuaja.

/* Ky është një koment në CSS */ h1 { color: blue; background-color: lightgray; }

Fontet (Shkrimi)

Mund të zgjidhni familjen e shkrimit (psh. Arial, Times New Roman), madhësinë dhe trashësinë (bold, italic).

p { font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; }

Struktura e Kutive

Çdo element HTML është një "kuti". Përdorni margin për hapësirën e jashtme dhe padding për atë të brendshme.

.kutia { margin: 20px; padding: 15px; border: 1px solid black; }
